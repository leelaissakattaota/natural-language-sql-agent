## 📌 Project Overview
Natural Language to SQL agent using **IBM Granite 3.2 8B** and LangChain's `create_sql_agent`. Ask questions in plain English about the Chinook music database and get SQL queries + results automatically. Demonstrates Text-to-SQL capabilities with real database interaction.

**Domain:** NLP → SQL — Database Intelligence  
**LLM:** ibm/granite-3-2-8b-instruct (IBM Watsonx)  
**Database:** Chinook MySQL (music store: artists, albums, tracks, customers)

## 🛠️ Tech Stack
| Component | Technology |
|---|---|
| LLM | IBM Granite 3.2 8B (Watsonx) |
| SQL Agent | LangChain `create_sql_agent` |
| Database | MySQL (Chinook) via SQLAlchemy |
| LangChain | langchain-community SQLDatabase |

## 🚀 Key Features
- Natural language → SQL query generation
- Auto-executes queries and returns results
- Chinook DB: 11 tables (artists, albums, tracks, customers, invoices)
- LangChain SQL toolkit integration
- IBM Watsonx LLM for query generation
